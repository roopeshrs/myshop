# Ecommerce Platform

## Getting Started
- Clone the project
- Go to the project directory
- Install dependencies >> npm install / npm i
- Start the server >> npm start

## Features
- Category and image carousel
- Infinite scroll
- Search with debouncing
- Filter
- Routing
- Responsive design

## Technologies used
- React
- Redux
- Tailwind Css
- Material UI
